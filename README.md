## 🧠 About

Ntali Games is an indie game development studio created by Nickolas, a Computer Engineering student and solo developer.

This repository contains the source code and content for the official Ntali Games website, where game updates, visuals, and development progress are showcased.

## 🎮 Current Project — Rush Track

**Rush Track** is an upcoming PC game inspired by the classic *Rail Rush*, reimagined with modern gameplay, improved visuals, and a fresh experience.

### 🔹 Vision
- Fast-paced gameplay  
- Smooth mechanics  
- Immersive environments  
- Modernized take on a classic concept  

### 🔹 Platform
- 💻 PC (primary focus)
